Unidade 01

Tema do Curso

O minicurso terá como propósito apresentar o conceito e a importância da ferramenta SQL (Structured Query Language), ou seja, Linguagem de Consulta Estruturada, que é uma linguagem utilizada para gerenciar, consultar e manipular dados armazenados em bancos de dados relacionais.

Esta linguagem foi criada por engenheiros da IBM na década de 1970. O SQL foi a primeira linguagem de DBMS (Sistema de Gerenciamento de Banco de Dados) considerada padrão em comunicação de banco de dados relacional pelo ANSI (American National Standards Institute) e pela ISSO (International Organization for Standardization).

O SQL é universalmente reconhecido pela sua robustez e versatilidade. Sua ampla adoção em vários setores e plataformas ressalta sua confiabilidade e eficiência, se tornando parte fundamental de muitas aplicações web e aplicativos de celular.

É importante enfatizar que o SQL, ao invés de ser uma linguagem de programação (ex.: C#, Java, Python, etc) para construir sites, apps ou programas, é uma linguagem que permite que aplicações criem, recuperem e manipulem dados de bancos de dados, e que os profissionais de tecnologia extraiam informações relevantes, façam análises complexas e tomem decisões embasadas em dados sólidos.

Para qualquer profissional que pretenda trabalhar com bancos de dados, dominar o SQL não é apenas uma recomendação, é uma ótima escolha, pois facilita o gerenciamento e a análise de dados contínuos em diversos sistemas, o que torna, portanto, o SQL uma ferramenta fundamental na área de tecnologia, desempenhando um papel essencial na carreira dos profissionais desta área.



Objetivos Gerais e Específicos do Curso

Objetivos Gerais: O objetivo geral deste curso EAD de curta duração é fornecer, gratuitamente, conhecimentos básicos para que o discente, ao final do curso, seja capaz de compreender os conceitos básicos da linguagem SQL e colocá-los em prática, testando suas habilidades, por meio de um questionário aplicado ao término de cada unidade estudada, fomentando para que o discente busque uma evolução para um novo estágio de conhecimento, a fim de prover fundamentos e técnicas necessárias para trabalhar com bancos de dados relacionais.

Objetivos Específicos:
- Introdução a linguagem SQL;
- Introdução a Sistema Gerenciador de Banco de Dados (SGBD);
- Apresentação dos principais comandos para definição de estruturas e manipulação de dados (SQL/DML/DDL/DCL/DTL);



Ementa do Curso

Será oferecido material teórico e prático com abordagens, estratégias e técnicas, a fim de propor uma série de tópicos relacionados ao tema a ser trabalhado ao longo do curso.
Na primeira unidade trabalharemos a parte teórica, utilizando a plataforma da Unifenas-MG, adotando a seguinte estrutura:
1. Introdução à Linguagem SQL;
2. Introdução a Sistema Gerenciador de Banco de Dados (SGBD);
3. Questionário com 5 perguntas.

Na segunda unidade trabalharemos a parte Prática, utilizando a plataforma da Unifenas-MG, adotando a seguinte estrutura:
1. Apresentação dos comandos para definição de estruturas e manipulação de dados (DQL/DML/DDL/DCL/DTL);
2. Criação de um banco de dados para um caso específico;
3. Questionário com 5 perguntas.
